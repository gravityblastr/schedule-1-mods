# GravityBlastr Modpack

All [GravityBlastr](https://thunderstore.io/c/schedule-i/p/gravityblastr/) mods for Schedule I in a single install.

## Included mods

- **[DeliveryNotificationsMod](https://thunderstore.io/c/schedule-i/p/gravityblastr/DeliveryNotificationsMod/)** — Rich delivery arrival/completion notifications with store name, destination, and loading dock
- **[DontPunchRayMod](https://thunderstore.io/c/schedule-i/p/gravityblastr/DontPunchRayMod/)** — Adds a "Sell" option to Ray and Jeremy's dialog so you don't have to punch them
- **[IlluminationMod](https://thunderstore.io/c/schedule-i/p/gravityblastr/IlluminationMod/)** — Extended flashlight and headlight range, plus sticky reverse lights
- **[LaunderMaxMod](https://thunderstore.io/c/schedule-i/p/gravityblastr/LaunderMaxMod/)** — Auto-selects the maximum amount in the laundering UI
- **[LaunderScaleMod](https://thunderstore.io/c/schedule-i/p/gravityblastr/LaunderScaleMod/)** — Scales laundering capacity with player rank
- **[LightPersistMod](https://thunderstore.io/c/schedule-i/p/gravityblastr/LightPersistMod/)** — Saves flashlight and headlight state across loads
- **[LockerNotificationsMod](https://thunderstore.io/c/schedule-i/p/gravityblastr/LockerNotificationsMod/)** — Alerts when employee pay lockers are running low
- **[PrefsPersistMod](https://thunderstore.io/c/schedule-i/p/gravityblastr/PrefsPersistMod/)** — Fixes settings not persisting on certain system locales
- **[SigningFreeMod](https://thunderstore.io/c/schedule-i/p/gravityblastr/SigningFreeMod/)** — Fixes the "signing free" typo to "signing fee"
- **[UnpackMod](https://thunderstore.io/c/schedule-i/p/gravityblastr/UnpackMod/)** — Allows brick unpacking when the packaging slot is occupied

## Compatibility

All mods support both IL2CPP and Mono branches of the game. See individual mod pages for details.
